[Litewrite Chrome App]

unobtrusive writing ported as a Chrome native app by Alex Neri

Seeing as this is an app that wonderfully does its job as a writing tool, I believe that it can be used as a native desktop app, though the use of the Chrome App framework. It's a small project that I plan to work on little by little, so there's no release date yet ^^;

This project is a work built on the original Litewrite code made by Jan-Christoph Borchardt & Jorin Vogel

Inspired by iA Writer & LightWrite. Initial prototype built on Notes. Alegreya typeface by Juan Pablo del Peral

Remotestorage

This is an unhosted web app, meaning its code is fully client-side, without any server backend you need to trust! It also supports the open remotestorage protocol so you can sync your data across devices & browsers.
